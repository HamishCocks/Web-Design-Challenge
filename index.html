Summary: Latitude vs X


The purpose of this project was to analyse how weather patterns change as one nears the equator. Data was pulled from the OpenWeatherMap API to generate a set of over 500 cities from around the world. The dataset was then cleaned and transformed using the Python library Pandas as well Matplotlib to generate graphs that plotted against several weather conditions/variables.


We have selected 4 to present on this site: Maximum Temperature, Humdiity, Cloudiness, Wind Speed. We provide a page in which the viewer can see each plot compared to the other, as well as more in-depth analyses of the correlations and patterns found in each analysis. We also provide the raw data that was used for the analysis.
